# House-Price-Prediction

Machine Learning project for predicting house prices using data preprocessing, exploratory data analysis (EDA), and regression techniques in Python.

## 📌 Project Overview

This project aims to predict house prices based on various housing features using Machine Learning algorithms. The project includes data preprocessing, exploratory data analysis (EDA), model building, training, and evaluation.

## 🎯 Objectives

* Analyze housing data and identify important features.
* Perform data cleaning and preprocessing.
* Visualize data using EDA techniques.
* Build and train regression models.
* Evaluate model performance and predict house prices.

## 📊 Dataset

The project uses the `Housing.csv` dataset containing various house features such as:

* Area
* Number of Bedrooms
* Number of Bathrooms
* Parking Availability
* Furnishing Status
* Main Road Access
* Air Conditioning
* Price of the House

## 🤖 Machine Learning Workflow

1. Import the dataset
2. Data preprocessing and cleaning
3. Exploratory Data Analysis (EDA)
4. Feature engineering and encoding
5. Train-test split
6. Model training using regression algorithms
7. Model evaluation
8. House price prediction

## 📈 Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/vinaygadde2020/House-Price-Prediction.git
```

### 2. Navigate to the Project Directory

```bash
cd House-Price-Prediction
```

### 3. Open the Notebook

```bash
jupyter notebook House_Price_Prediction_Project.ipynb
```

Or upload `House_Price_Prediction_Project.ipynb` and `Housing.csv` to Google Colab and run all cells.

The trained regression model can estimate house prices based on input housing features and demonstrates the practical application of Machine Learning in real-estate price prediction.
